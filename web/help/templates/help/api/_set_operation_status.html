Set the Status of an Operation

POST {server}/api/set_operation_status/

Set the status of a specified non COMPLETED operation. If successful, returns the operation details.

States:

  • OPEN
  • CLOSED
  • COMPLETED
ONCE AN OPERATION IS SET TO COMPLETED, IT'S ONLY POSSIBLE TO CONSULT IT.
WILL NO LONGER POSSIBLE TO EDIT ANY PART OF THE OPERATION!
TO CHANGE THE STATUS OF A COMPLETED OPERATION, AN ADMINISTRATOR MUST CHANGE IT DIRECTLY IN THE DATABASE OR THROUGH THE ADMINISTRATIVE PANEL.

HEADER PARAMETERS:
Authorization
-
Token [token]

REQUEST BODY SCHEMA:
operation_id
number
Operation ID
status
string
Status (OPEN | CLOSED | COMPLETED)
